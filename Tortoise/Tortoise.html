Pen Up:
Stops the drawing line, allowing the turtle to move without drawing a trail.
Command Syntax: pen_up

Pen Down:
Starts or resumes drawing as the turtle moves.
Command Syntax: pen_down

Forward:
Moves the turtle forward by the specified distance.
Command Syntax: forward (distance)

Turn:
Rotates the turtle by a specified angle.
Command Syntax: turn (angle)

Draw Polygon:
Draws a polygon with a specified number of sides and side length.
Command Syntax: polygon (number of edges)

Draw Complete Graph:
Creates a complete graph with specified number of edges and length
Command Syntax: graph (number of edges) (length of edge)

Draw Koch Curve:
Draws a Koch fractal curve based on the number of edges, distance, and limit.
Command Syntax: koch (number of edges) (length of edge) (limit)

Draw Sierpinski Triangle:
Creates a Sierpinski Triangle fractal to the specified limit.
Command Syntax: sierpinski (length of edge) (limit)

Reset canvas:
Resets the canvas to its default state
Command Syntax: reset
